What is XML?

  • XML stands for EXtensible Markup Languague
  • XML is a markup languague much like HTML
  • XML is designed to describe data
  • XML are not predefined. You must define your own tags
  • XML uses a Document Type Definition (DTD) or an XML Schema to describe the data
  • XML with a DTD or XML Schema is designs to be self-descriptive

The main diference between XML and HTML

XML was designed to carry data.

XML is not a replacement for HTML.

XML and HTML were designed with different goals:

XML was designed to describe data and to focus in what data is.

HTML was designed to display data and focus how data looks.

HTML is about displaying information, while XML is about describing information.
